# DecodeLabs Internship Project 1

A responsive layout project for a full-stack development training platform.

## Project Structure

```
DecodeLabs-Internship-Project-1/
├── aesthetics.html
├── architecture.html
├── conclusion.html
├── css/
│   └── styles.css
├── framework.html
├── guidelines.html
├── index.html
├── js/
│   └── main.js
└── toolkit.html
```

## Pages

- **index.html** - Welcome and introduction page
- **guidelines.html** - Project guidelines and criteria
- **architecture.html** - Project architecture overview
- **toolkit.html** - Development toolkit
- **aesthetics.html** - Design system and aesthetics
- **framework.html** - Project roadmap and framework
- **conclusion.html** - Conclusion and next steps

## Features

- Responsive design (mobile, tablet, desktop)
- Material Design icons
- Clean, modern UI with bento grid layouts
- Mobile drawer navigation
- Sidebar navigation (desktop)
- Smooth animations and transitions
- Accessibility features

## Getting Started

To view the project locally, simply open `index.html` in your browser.

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom design system with CSS variables
- **Vanilla JavaScript** - Interactive features
- **Material Icons** - Icon library
- **Google Fonts** - Montserrat (headings) and Open Sans (body)
